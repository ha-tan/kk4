工工四作成ツール kk4
====================

はじめに
--------
  三線の工工四を作成するツールです。

使い型
------
  所定のフォーマットで記載したテキストファイルを
  kk4.exeにドラッグ & ドロップしてください。

  フォーマットの詳細はサンプルファイル(sample.txt)を参照してください。
